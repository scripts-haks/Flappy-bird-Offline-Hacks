# 🐦 Flappy Bird Offline Hacks

A collection of browser console scripts that inject a cheat menu into the **Flappy Bird Offline** Chrome extension.

## 📋 How To Use

1. Install [Flappy Bird Offline](https://chromewebstore.google.com/detail/flappy-bird-offline/pidkjcapccmhpdokcjnkngcflobhopdl) from the Chrome Web Store
2. Open the extension and go full screen mode and start a game
3. Open Chrome DevTools (`F12` or `Right Click > Inspect`)
4. Go to the **Console** tab
5. Paste the script and hit **Enter**
6. A cheat menu will appear in the top right corner

## 🛠️ Features

| Feature | Description |
|---|---|
| 👻 Ghost Pipes | Fly straight through pipes |
| 🚀 Ludicrous Speed | Pipes fly across the screen at insane speed |
| 🐢 Slow Motion | Slow everything right down |
| 🪄 Double Pipes | Spawns two pipes at once |
| 🤖 Auto Flap | The bird plays itself |
| ❄️ Freeze Pipes | Pipes stop mid-air |
| 💰 Auto Score | Score ticks up automatically |
| 🎯 Set Score | Set your score and highscore to any number |
| 📏 Pipe Gap | Change how big the gap between pipes is |
| 🔢 Pts Per Pipe | Set how many points each pipe gives |
| 🐦 Bird Colour | Change the colour of the bird |
| 🔄 Reset | Restore everything back to normal |

This project is for **personal and educational use only**. It modifies the behaviour of a third party Chrome extension by injecting JavaScript via the browser console. Use it responsibly.
